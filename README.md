# Mixa din drink

### Detta är ett enkelt spel där du själv får mixa din favoritdrink.

Du får själv välja i vilken miljö du känner att du vill dricka din drink.  
Hemma? På stranden? Du väljer själv.

Därefter är det dags att göra dina dryckesval. Beroende på val, byts musik till passande plats.
Snart har du din drink färdig och redo att avnjutas på platsen du har valt!
